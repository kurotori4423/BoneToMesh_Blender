# BoneToMesh_Blender
ボーンから動作検証用のメッシュを作成するBlender2.79アドオン

## インストール  
「Blenderユーザー設定」 > 「ファイルからアドオンをインストール...」　から「bone_to_mesh.py」を選択しインストール  
チェックを入れてアドオンを有効にする。

## 使用方法
メッシュを作成したいアーマチュアを選択し、「編集モード」に移行、
ボーンすべてを選択状態にして、メニューから「追加」 > 「選択ボーンからメッシュ」を選択。
